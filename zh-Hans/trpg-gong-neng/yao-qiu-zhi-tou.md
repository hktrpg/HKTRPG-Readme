# 要求掷骰

![使用示范](<../.gitbook/assets/image (26).png>)

### 要求掷骰功能

方便GM提示玩家需要进行的掷骰，支持角色卡，所以可以调用.ch功能。

输入指令后，会弹出按钮，点击后就会自动进行掷骰。

### 使用方法

`.re 要求掷骰`\
例如 .`re 1d100 哈哈, 1d3 SC成功, 1d10 SC失败`

每个掷骰方法需要使用逗号(,)进行分割

{% hint style="info" %}
注意: 本功能支持Discord斜线(/)功能，只需要把(.)转成(/)
{% endhint %}

可与 [按钮转发 `.forward`](../xi-tong-gong-ju/discord-an-niu-zhuan-fa.md) 搭配，把点击结果转到指定频道。
