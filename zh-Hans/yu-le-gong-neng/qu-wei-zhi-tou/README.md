# 趣味掷骰

玩乐用指令，如**塔罗牌**、**运势**、**随机抽选**

* `排序 (选项1) (选项2) (选项N)` 会进行随机的排序
* `随机 (选项1) (选项2) (选项N)` - 会随机抽选其中一个出来
* choice `(选项1) (选项2) (选项N)` - 会随机抽选其中一个出来\
  ![](<../../.gitbook/assets/image (18).png>)
* `立flag` 会随机出现一些立FLAG的场景\
  ![](<../../.gitbook/assets/image (41).png>)
* `.me` 可以令HKTRPG机械人重复你的说话
* `.mehistory`  显示在该频道的过往.me发言

{% hint style="info" %}
因为会互相冲突，.me 的Discord 斜线指令(/)是 **`/mee`**
{% endhint %}
