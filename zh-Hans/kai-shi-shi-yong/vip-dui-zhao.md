# VIP 权益对照

查询自己／群组等级：输入 `.patreon level` 或 `.admin id`。

赞助与方案说明见 [开发支持](kai-fa-zhi-yuan.md)。

{% hint style="warning" %}
下表为文档整理用摘要（对应 Bot 内各模块的等级数组）。服务器可能调整上限；冲突时以 Bot 即时提示为准。
{% endhint %}

### 等级概念

* **VIP 0**：一般用户
* **VIP 1+**：Patreon／Boost 等提升后的等级（个人或群组取较高者）
* 部分功能同时看「个人 VIP」与「群组 VIP」

### 常见功能上限（摘要）

| 功能 | VIP 0（约） | VIP 1～2（约） | VIP 3+（约） | 说明 |
| ---- | ----------- | -------------- | ------------ | ---- |
| 角色卡数量 | 4 | 20 | 30～99 | `.char` |
| 按钮转发 `.forward` | 4 | 20 | 30～99 | Discord |
| 自定义骰子 `.ra`（群组） | 30 | 200 | 500 | 群组关键字 |
| 自定义骰子（个人） | 2 | 200 | 500 | 个人关键字 |
| 数据库 `.db` | 30 | 200 | 300 | |
| 保存指令 `.cmd` | 30 | 200 | 300 | |
| 定时 `.at` | 5 | 25～50 | 200 | |
| 周期 `.cron` | 2 | 15～30 | 45～99 | |
| Discord 团录导出（周） | 1 | 20～40 | 40～99 | 另有冷却 |
| StoryTeller 剧本数 | 3 | 10 | 100 | 见 StoryTeller 页 |
| StoryTeller 同时游戏 | 3 | 10 | 100 | |
| AI 对话／翻译字数 | 较低 | 较高 | 更高／解锁中级模型 | 见 AI 页 |
