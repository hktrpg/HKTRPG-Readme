# 开始进行TRPG

TRPG是一类非常好玩的游戏模式，可能你被它的游戏性或是自由度所吸引

它通常由几种元素组成，包括

* 随机性(抽牌，掷骰)
* 数值(角色的能力，属性，挑战的难度)
* 数据(世界观，书籍，文章)

所以需要不同的工具去纪录和处理它们，在面团中最基本是实物的骰子，纸笔墨，但在网络中你可以使用电子的方式进行取代。

在下面会介绍不同的工具去取代实物。
