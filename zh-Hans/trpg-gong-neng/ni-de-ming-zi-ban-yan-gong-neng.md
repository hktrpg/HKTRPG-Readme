# 你的名字(扮演功能)

{% hint style="info" %}
在Discord之中，你在群组中的名字可以随时改变，但问题是这种改变是追溯整个群组的，导致如果你有使用文本跑团，角色的名字图片将会被打乱。

请注意，这种方式只可以以[.edit (修改旧消息功能)](../xi-tong-gong-ju/discord-xiu-gai-jiu-xun-xi.md) 来**追加修改内容**。
{% endhint %}

`.myname` `.me` `.me1` `.me(名字)` \*Discord限定功能

### TRPG扮演发言功能

你可以设置一个角色的名字及头像，

然后你只要输入指令和说话，就会帮你使用该角色发言。

支持掷骰，请使用`[[]]`来包着掷骰指令 如`[[1d100]]`

注意: 此功能需求编辑Webhook及消息功能，请确定授权

### 使用教学

#### 1.设置角色

输入 `.myname "名字" 角色图片网址 名字缩写(非必要)`

例子 `.myname "泉心 造史"` [`https://images.pexels.com/photos/10013067/pexels-photo-10013067.jpeg`](https://images.pexels.com/photos/10013067/pexels-photo-10013067.jpeg) `造`

_**名字**_&#x662F;角色名字，会作为角色显示的名字，但如果该名字有空格就需要用开`引号"`包着

&#x5982;**"泉心 造史"** 不然可以省去`"`

_**图片**_&#x5247;是角色图标，如果图片出错会变成最简单的Discord图标

图片可以直接上传到DISCORD或IMGUR.COM上

_**名字缩写**_&#x662F; 是用来方便你启动它

例&#x5982;_`.me造 「来玩吧」`_

#### 2.删除角色

_**`.myname delete 序号 / 名字缩写 / "名字"`**_

删除方式是delete 后面接上序号或名字缩写或名字

#### 3.显示角色

`.myname show`

#### 4.使用角色

**`.me(序号/名字缩写) 消息`**

如

* **`.me1 泉心慢慢的走到他们旁边，伺机行动`**
* ![](<../.gitbook/assets/image (13).png>)
* **`.me造 「我接受你的挑战」`**

### 功能一覧

* `.myname` - 设置角色
* `.myname delete 序号 / 名字缩写` - 删除角色
* `.myname show` - 显示角色列表
* `.me(序号/名字缩写) 消息` - 使用角色发言
* `.mehistory` - 显示该频道过往.me发言
