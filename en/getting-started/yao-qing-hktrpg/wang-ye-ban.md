# Web Version

### Creating a Room

[HKTRPG Web Version](https://rollbot.hktrpg.com/)

The web chat is organized by rooms. On entry you default to the **Public Room**. Change the name on the right to join another room. The name on the left is your display name in the chat—you can change that freely too.

![](<../../.gitbook/assets/image (47).png>)

{% hint style="info" %}
Designed for simplicity: any room name grants access—there are no passwords. Each room holds up to 100 messages; older ones are removed automatically.
{% endhint %}

![](<../../.gitbook/assets/image (58).png>)

The right side shows your current location and the total number of people across all rooms.<br>

![](<../../.gitbook/assets/image (56).png>)

The center shows chat messages; the input bar below accepts chat and roll commands.

{% hint style="success" %}
Closing the page logs you out, but chat history remains until the message limit is reached.
{% endhint %}
