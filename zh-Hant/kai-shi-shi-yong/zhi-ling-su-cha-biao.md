# 指令速查表

在任意平台對 HKTRPG 輸入 `bothelp`，可依分類瀏覽；本頁整理最常用指令。

| 分類指令 | 內容 |
| -------- | ---- |
| `bothelp` | 總覽 |
| `bothelp Base` | 基本擲骰、暗骰、角色卡等 |
| `bothelp Dice` | 各系統骰組列表 |
| `bothelp Tool` | 輔助工具 |
| `bothelp admin` | 管理工具 |
| `bothelp funny` | 娛樂功能 |
| `bothelp link` | 平台邀請連結 |
| `bothelp privacy` | 隱私權 |

完整線上說明亦可參考：<https://bothelp.hktrpg.com>
