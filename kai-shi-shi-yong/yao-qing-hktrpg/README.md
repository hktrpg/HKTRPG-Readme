# 邀請HKTRPG

## 邀請連結

本頁包含關於如何邀請並說用HKTRPG說明。

使用HKTRPG，你既可以經網頁使用，也可以經不同聊天平台來使用。

然而，使用聊天平台如Discord，Line等等的話，可以新增個人化/頻道化數據並在頻道中進行管理。
