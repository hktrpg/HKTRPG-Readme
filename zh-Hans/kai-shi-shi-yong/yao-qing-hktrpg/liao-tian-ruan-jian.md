# 聊天软件

### 邀请

通常来说，聊天软件的机械人都有两种模式，一是直接和机械人聊天，一是把机械人邀请到私人的群组之中。

而对于HKTRPG来说，你可以在两种模式中使用大部份功能，只是某些功能只可以在群组中使用。

{% hint style="info" %}
只限在群组使用的功能如暗骰功能，自定骰子功能，数据库功能。\
亦有Discord限定功能，如角色扮演功能，聊天纪录导出功能。\\
{% endhint %}

ID: HKTRPG#4989

在Discord -> Home -> Find or start a conversation 中输入ID就可以使用HKTRPG了\
不过邀请到群组的话还是需要使用下面的链接\
[https://discord.hktrpg.com](https://discord.hktrpg.com)\

也可以按+朋友。\
接着在群组中就可以像一般朋友那样邀请他加入，但要注意LINE限定一个群组只可以加入一个机械人。<br>
