# 自定义骰子

这是根据关键字来随机抽选功能,只要符合内容,以后就会随机抽选

{% hint style="info" %}
本功能已改版， .rap 转成个人专用的骰组， \
原全服群组(.rap)变成 .ras \
.ra => random answer (group) \
.rap => random answer personal \
.ras => random answer server
{% endhint %}

### 使用方法

准备骰子内容，及骰子的名字

在群组中输入`.ra add (骰子名称) (骰子内容1 2 3 4 5)`\
这时应该添加成功，然后你可以输入

`.ra (骰子名称)`来投掷此骰子

### 功能一覧

`.ra(p|s)(次数) (add del show 骰子名称)`

* 输入.ra add (`骰子名称`) (选项1) (选项2) (选项3)即可增加关键字&#x20;

&#x20;     重复输入，可以增加选项，总共上限2000字

* 输入.ra show 显示所有`骰子`及 编号&#x20;
* 输入.ra show (`骰子名称`)显示内容&#x20;
* 输入.ra del (`骰子名称`) 即可删除&#x20;
* 输入.ra(次数,最多30次) (`骰子名称`1/编号) (`骰子名称`2)...(`骰子名称`n)&#x20;

&#x20;     即可不重复随机抽选&#x20;

* 输入.rra(次数,最多30次) (`骰子名称`1/编号)(`骰子名称`2)...(`骰子名称`n)&#x20;

&#x20;     即可重复随机抽选&#x20;

&#x20; `骰子名称`可用数字代替, 如编号5,可以轮入 .ra 5&#x20;

### 三种模式

* 如使用.ra 是群组版, 供整个群组共用&#x20;
* 如使用.ras 是公开版, 在整个HKTRPG可以看到&#x20;
* 如使用.rap 是个人专用版, 只有自己可以使用
* 例如输入 .ras10 圣晶石召唤 即可十连抽了

### 代码

* 亦支持代码，输入.ras newType 可以观看效果
* {br} <--隔一行
* {ran:100} <---随机1-100
* {random:5-20} <---随机5-20
* {server.member\_count} <---现在频道中总人数
* {my.name} <---显示掷骰者名字

以下需要开启.level 功能

* {allgp.name} <---随机全GP其中一人名字
* {allgp.title} <---随机全GP其中一种称号
* {my.RankingPer} <---现在排名百分比
* {my.Ranking} <---显示掷骰者现在排名
* {my.exp} <---显示掷骰者经验值
* {my.title} <---显示掷骰者称号
* {my.level} <---显示掷骰者等级
