# HKTRPG  官方使用教学

## **欢迎来到HKTRPG!**

HKTRPG是一个用于TRPG的网上工具，它允许你在不同平台上(如:网站，Discord，Line，Telegram，Plurk，Whatsapp)上进行掷骰。而且它不仅可以用于TRPG，还可以用于日常娱乐及群组管理。

### 使用理由

作为一个GM，由创建一个群组，保持良好的活跃气氛，教玩家开卡掷骰，再到跑团并向玩家展示各种信息，都是一项艰辛的工作。&#x20;

而使用HKTRPG，你可以在不同的平台中使用统一的指令，没有转换的烦恼并自带娱乐功能。

它的设计就是为了减少你的工作!&#x20;

你可以边使用HKTRPG边看说明!
