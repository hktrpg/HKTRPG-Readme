# 保存掷骰指令

`.cmd (add edit del show 自定关键字)`

* 这是根据关键字来再现掷骰指令,
* 例如输入 `.cmd add pc1斗殴 cc 80 斗殴`
* 再输入`.cmd pc1斗殴` 就会运行后方的指令
* add 后面第一个是关键字, 可以是符号或任何字
* P.S.如果没立即生效 用.cmd show 刷新一下
* 输入`.cmd add (关键字) (指令)`即可增加关键字
* 输入`.cmd edit (关键字) (指令)` 即可修改指令
* 输入`.cmd show` 显示所有关键字内容及编号
* 输入`.cmd del(编号)`或all 即可删除
* 输入`.cmd (关键字/编号)` 即可运行
