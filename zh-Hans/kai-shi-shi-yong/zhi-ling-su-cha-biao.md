# 指令速查表

在任意平台对 HKTRPG 输入 `bothelp`，可依分类浏览；本页整理最常用指令。

| 分类指令 | 内容 |
| -------- | ---- |
| `bothelp` | 总览 |
| `bothelp Base` | 基本掷骰、暗骰、角色卡等 |
| `bothelp Dice` | 各系统骰组列表 |
| `bothelp Tool` | 辅助工具 |
| `bothelp admin` | 管理工具 |
| `bothelp funny` | 娱乐功能 |
| `bothelp link` | 平台邀请链接 |
| `bothelp privacy` | 隐私权 |

完整线上说明亦可参考：<https://bothelp.hktrpg.com>
