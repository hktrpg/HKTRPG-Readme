# 平台差異表

HKTRPG 多數擲骰指令在 Discord、LINE、Telegram、WhatsApp、Plurk、網頁皆可使用；下列功能有平台限制。

### 功能 × 平台

| 功能 | Discord | LINE | Telegram | WhatsApp | Plurk | 網頁 |
| ---- | ------- | ---- | -------- | -------- | ----- | ---- |
| 基本／系統擲骰 | ✓ | ✓ | ✓ | ✓（不穩） | ✓ | ✓ |
| 暗骰 `dr` / `ddr` | ✓ | ✓ | ✓ | △ | △ | △ |
| 角色卡 `.char` / `.ch` | ✓ | ✓ | ✓ | △ | △ | ✓（可連動） |
| 角色卡按鈕 `.ch button` | ✓ | — | — | — | — | — |
| 要求擲骰 `.re` | ✓ | — | — | — | — | — |
| 按鈕轉發 `.forward` | ✓ | — | — | — | — | — |
| 扮演 `.myname` / webhook | ✓ | — | — | — | — | — |
| 修改舊訊息 `.edit` | ✓ | — | — | — | — | — |
| 團錄匯出 `.discord` | ✓ | — | — | — | — | — |
| 身分組 Reaction | ✓ | — | — | — | — | — |
| 斜線指令 `/` | ✓ | — | — | — | — | — |
| AI 翻譯檔案／長文 | ✓ 為主 | △ | △ | △ | △ | △ |

✓ 支援　— 不支援　△ 有限制或不穩定

### 其他注意

* **LINE**：一個群組通常只能加入一個機械人
* **WhatsApp**：連線較不穩定，可能暫時失效
* **群組限定**：暗骰 GM、自定骰、資料庫等部分功能需在群組使用
