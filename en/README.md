# HKTRPG Official User Guide

### **Welcome to HKTRPG!**

HKTRPG is an online toolkit for TRPG. It lets you roll dice across platforms (website, Discord, LINE, Telegram, Plurk, WhatsApp). Beyond TRPG, it also works for everyday fun and group management.

#### Why use it

As a GM, building a group, keeping the vibe lively, teaching players how to make characters and roll dice, then running sessions and sharing info with players is hard work.

With HKTRPG, you use the same commands across platforms—no switching overhead—plus built-in entertainment features.

It is designed to cut your workload!

#### Let’s get started!

This site has a lot of information. You don’t need to know it all.\
Use HKTRPG while you read the docs!

**Fun fact:** HKTRPG is a native Traditional Chinese project. It started for running CoC, and now independently supports a dozen+ TRPG systems. If you like it, you can [support on Patreon](https://www.patreon.com/HKTRPG).
