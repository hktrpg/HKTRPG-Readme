# 基本掷骰

{% hint style="info" %}
小知&#x8B58;**:** 如果看其他TRPG/桌游教学的话，最普遍的骰子表示方式是xDy，代表有x粒y面的骰子\
例如去酒吧玩的吹牛(大话骰)就是大家掷五粒六面的骰子，即5D6。
{% endhint %}

### xDy

掷出x粒y面的骰子，\
如1D100 这指令代表掷出一粒一百面的骰子，结果将会是1-100 &#x20;

又例如输入 1D100 攻撃！ \
会输出     1D100：攻撃！  38\[38] = 38

![](<../../.gitbook/assets/image (33) (1) (1).png>)\
如上面一样,在骰子数字后方隔空白位打字,可以进行发言。

### 支持kh|kl|dh|dl

{% hint style="info" %}
这种掷骰要求常见于DND，PF
{% endhint %}

k keep保留，d drop 放弃，h highest最高，l lowest最低 \
如3d6kh \
保留最大的1粒骰\
![](<../../.gitbook/assets/image (29).png>)\
\
3D6dl2\
放弃最小的2粒骰\
![](<../../.gitbook/assets/image (26) (1).png>)
