# HKTRPG  官方使用教學

## **歡迎來到HKTRPG!**

HKTRPG是一個用於TRPG的網上工具，它允許你在不同平台上(如:網站，Discord，Line，Telegram，Plurk，Whatsapp)上進行擲骰。而且它不僅可以用於TRPG，還可以用於日常娛樂及群組管理。

### 使用理由

作為一個GM，由建立一個群組，保持良好的活躍氣氛，教玩家開卡擲骰，再到跑團並向玩家展示各種信息，都是一項艱辛的工作。&#x20;

而使用HKTRPG，你可以在不同的平台中使用統一的指令，沒有轉換的煩惱並自帶娛樂功能。

它的設計就是為了減少你的工作!&#x20;

### 5 分鐘上手

1. **邀請** HKTRPG 到你的平台 → [邀請HKTRPG](kai-shi-shi-yong/yao-qing-hktrpg/README.md)
2. （可選）切換回覆語言：`.lang zh-tw`／`.lang zh-hans`／`.lang en`（詳見 [語言設定](kai-shi-shi-yong/yu-yan-she-ding-duo-yu-xi.md)）
3. 輸入 `1d100` 試試基本擲骰
4. 輸入 `bothelp` 查看指令分類（或看 [指令速查表](kai-shi-shi-yong/zhi-ling-su-cha-biao.md)）
5. 建立 [角色卡](trpg-gong-neng/kai-shi-jin-hang-trpg/jiao-se-ka.md)：`.char add …` / `.char use 名字`
6. 需要隱藏結果時用 [暗骰](trpg-gong-neng/kai-shi-jin-hang-trpg/an-tou.md)：`dr` / `ddr` / `dddr`

{% hint style="info" %}
**小知識:** HKTRPG是原生的正體中文程式，本來是為了跑CoC而開發，但現在獨立支援十數種TRPG系統。覺得做得好，可以[Patreon支持](https://www.patreon.com/HKTRPG)。
{% endhint %}

### 依 bothelp 四大區塊瀏覽

Bot 內輸入 `bothelp` 可分四類查詢；文件亦按此對齊：

| bothelp | 說明 | 文件入口 |
| ------- | ---- | -------- |
| `bothelp Base` | 基本擲骰、暗骰、角色卡等 | [開始進行TRPG](trpg-gong-neng/kai-shi-jin-hang-trpg/README.md) |
| `bothelp Dice` | 各 TRPG 系統骰組 | [指定TRPG系統](trpg-gong-neng/zhi-ding-trpg-xi-tong/README.md) |
| `bothelp Tool` / `admin` | 系統工具與管理 | [系統工具](xi-tong-gong-ju/gong-neng-kai-guan.md) |
| `bothelp funny` | 娛樂與日常功能 | [趣味擲骰](yu-le-gong-neng/qu-wei-zhi-tou/README.md) |
