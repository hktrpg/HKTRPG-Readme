# 平台差异表

HKTRPG 多数掷骰指令在 Discord、LINE、Telegram、WhatsApp、Plurk、网页皆可使用；下列功能有平台限制。

### 功能 × 平台

| 功能 | Discord | LINE | Telegram | WhatsApp | Plurk | 网页 |
| ---- | ------- | ---- | -------- | -------- | ----- | ---- |
| 基本／系统掷骰 | ✓ | ✓ | ✓ | ✓（不稳） | ✓ | ✓ |
| 暗骰 `dr` / `ddr` | ✓ | ✓ | ✓ | △ | △ | △ |
| 角色卡 `.char` / `.ch` | ✓ | ✓ | ✓ | △ | △ | ✓（可连动） |
| 角色卡按钮 `.ch button` | ✓ | — | — | — | — | — |
| 要求掷骰 `.re` | ✓ | — | — | — | — | — |
| 按钮转发 `.forward` | ✓ | — | — | — | — | — |
| 扮演 `.myname` / webhook | ✓ | — | — | — | — | — |
| 修改旧消息 `.edit` | ✓ | — | — | — | — | — |
| 团录导出 `.discord` | ✓ | — | — | — | — | — |
| 身份组 Reaction | ✓ | — | — | — | — | — |
| 斜线指令 `/` | ✓ | — | — | — | — | — |
| AI 翻译文件／长文 | ✓ 为主 | △ | △ | △ | △ | △ |

✓ 支持　— 不支持　△ 有限制或不稳定

### 其他注意

* **LINE**：一个群组通常只能加入一个机械人
* **WhatsApp**：连线较不稳定，可能暂时失效
* **群组限定**：暗骰 GM、自定骰、数据库等部分功能需在群组使用
