# TRPG Token制作功能

**`.token(2-3) [名字1] [名字2]` 制作TRPG Token**

**`.tokenupload` 上传Token到imgbox**

**Discord 限定**\
这是用来制作跑团Token的功能\
可以自定两行名字和图片内容 \
![](../.gitbook/assets/6cc9e2d9-dff5-43c4-9192-425d6876e928.jpg)

### 使用方法

Reply一个有图片的消息 或发送一张图片时，输入指令.token 或 .token2 就可以制作Token\
没有图片的话，会使用你的头像，作为token\
亦可以输入两行文本，作为图片上的名字 如

```
.token
Sad
HKTRPG
```

也可以不输入，那Token上就不会有文本

### 样式

{% hint style="info" %}
### 现在只有这三款格式，欢迎有创意/会画画的人，提供图样， 丰富这功能
{% endhint %}

`.token` 是一张即影即有相片 \
`.token2` 是一张圆型透明Token \
`.token3` 是一张根据输入名字随机外圈颜色的Token
