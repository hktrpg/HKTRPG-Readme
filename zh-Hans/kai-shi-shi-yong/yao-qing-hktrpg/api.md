# API

回复格式为JSON，请求方式为GET \
请求的位置: [https://api.hktrpg.com/](https://api.hktrpg.com/)

示范: [https://api.hktrpg.com/?msg=1d100](https://api.hktrpg.com/?msg=1d100)

![实际使用例子](https://github.com/hktrpg/TG.line.Discord.Roll.Bot/blob/master/views/image/api.png?raw=true)

{% hint style="info" %}
和网页版一样，这种方式不能输入ID，所以不能使用个人的数据库功能，如.ra .db 个人角色卡
{% endhint %}
