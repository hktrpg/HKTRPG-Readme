# 语言设置（多语系）

默认语言为**繁体中文（zh-tw）**。

{% hint style="info" %}
**小知识：** 切换语言只会改变机器人的**回复文本**。掷骰指令前缀不变，例如 `.cc`、`bothelp`、`/lang`、`.in` 等仍然相同。
{% endhint %}

### 支持语言

| 代码        | 语言       |
| --------- | -------- |
| `zh-tw`   | 繁体中文（默认） |
| `zh-hans` | 简体中文     |
| `en`      | English  |

{% hint style="warning" %}
请使用 `zh-hans` 切换简体，**不要**使用 `cn` 或 `zh-cn`（不支持）。
{% endhint %}

### 指令一览

| 指令                     | 说明                    |
| ---------------------- | --------------------- |
| `.lang` 或 `.lang show` | 显示目前语言                |
| `.lang list`           | 列出支持语言                |
| `.lang help`           | 显示说明                  |
| `.lang zh-tw`          | 设为繁体中文                |
| `.lang zh-hans`        | 设为简体中文                |
| `.lang en`             | 设为英文                  |
| `/lang`                | Discord Slash 版（功能相同） |

### 两种设置范围

语言设置分成两种范围，互不干扰：

#### 1. 服务器／群组（Server）

* **谁可以设：** 需要该群组的**管理员权限**
* **影响范围：** 此服务器／群组内，所有人看到的机器人回复
* **不影响：** 其他服务器、其他群组、任何人的私讯

范例（在 Discord 服务器或群组中输入）：

```
.lang en
.lang zh-hans
.lang zh-tw
```

或使用 Slash：

```
/lang
```

#### 2. 私讯（DM）

* **谁可以设：** 任何人
* **影响范围：** **只有你**与 HKTRPG 的私讯回复
* **不影响：** 任何服务器／群组的回复语言

范例（直接私讯 HKTRPG）：

```
.lang en
.lang zh-hans
.lang zh-tw
```

{% hint style="success" %}
想在团里用繁中、自己私讯查数据时用英文？可以：群组维持 `zh-tw`，私讯单独设 `.lang en`。
{% endhint %}

### 实际使用流程

#### 把整个 Discord 服务器改成英文

1. 确认你有该服务器的管理员权限
2. 在任一频道输入：

```
.lang en
```

3. 之后此服务器内的说明、错误消息、多数功能回复会改为英文
4. 要改回繁中：

```
.lang zh-tw
```

#### 只改自己的私讯语言

1. 打开与 HKTRPG 的私讯
2. 输入：

```
.lang zh-hans
```

3. 只有你的私讯会变成简体；群组语言不变

#### 查目前是什么语言

```
.lang show
```

或：

```
.lang
```

### 常见问题

#### Q：我改了私讯语言，为什么群组还是繁中？

这是预期行为。私讯语言与群组语言是分开保存的。

#### Q：我在群组输入 `.lang en`，为什么说没有权限？

群组／服务器语言需要**管理员权限**。一般成员只能在**私讯**设置自己的语言。

#### Q：Discord 接口语言会跟着变吗？

不会。`.lang` 控制的是 **HKTRPG 回复内容**的语言。\
Discord 客户端接口、以及 Slash 指令名称的本地化，仍依 Discord 本身的语言设置。

#### Q：指令会不会变成英文版（例如 `.cc` 变成别的）？

不会。指令前缀与骰组指令维持不变，只改回复文本。

#### Q：Line / Telegram / WhatsApp 也可以用吗？

可以。在对应平台的群组或私讯使用 `.lang` 即可（群组同样需要管理员权限才能改群组语言）。

#### Q：设置后没有效？

语言设置需要数据库连接才能保存。若系统暂时无法连接数据库，会提示无法保存；稍后再试即可。

### 快速对照

| 你想做的事     | 在哪里输入 | 指令              |
| --------- | ----- | --------------- |
| 全服务器用英文   | 服务器频道 | `.lang en`      |
| 全服务器用简体   | 服务器频道 | `.lang zh-hans` |
| 还原服务器繁中   | 服务器频道 | `.lang zh-tw`   |
| 只有我的私讯用英文 | 私讯    | `.lang en`      |
| 查看目前语言    | 任意处   | `.lang show`    |
| 看支持列表     | 任意处   | `.lang list`    |

{% hint style="info" %}
相关指令说明也可在聊天中输入 `.lang help` 或 `/lang` 查看。
{% endhint %}
