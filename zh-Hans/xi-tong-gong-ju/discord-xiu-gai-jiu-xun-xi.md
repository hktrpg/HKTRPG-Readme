# Discord 修改旧消息

{% hint style="info" %}
现在你可以修改由HKTRPG和webhook(角色扮演功能)所发出的消息\
就像你自己只能修改自己的消息一样，此功能不能修改其他人或其他BOT的消息。
{% endhint %}

`.edit`  Discord限定功能

![范例](<../.gitbook/assets/image (2) (2).png>)

### 使用方法:

对想要修改的信息右击点击reply \
然后按以下格式输入即可 \
`.edit 信息第一行` \
`信息第二行` \
`信息第三行`

{% hint style="info" %}
`使用前，`请确认自己拥有Admin 或频道管理权限。
{% endhint %}
