# 范例

以下是四个范例剧本的简介：

1.  **猫咪的一天**&#x20;

    这是一个轻松的角色扮演故事，玩家将扮演一只独自看家的猫，可以选择要当个乖巧的猫咪、恶作剧的小坏蛋，或是聪明的小帮手，并根据一天的行为迎来不同的结局 。
2.  **爱之语心理测验**&#x20;

    此为一个包含20个问题的心理测验，通过一系列的情境选择，帮助用户分析并找出自己在亲密关系中最主要的「爱之语」是属于哪一种类型 。
3.  **灵探事件：山中老村**&#x20;

    这是一个带有悬疑与恐怖元素的文本冒险游戏，玩家为了摆脱过去而前往一座深山古村，却被卷入一场围绕着神秘仪式与村民的诡异事件中 。
4.  **功能测试脚本**&#x20;

    这是一个用于技术展示与验证的剧本，其内核在于通过数十种复杂的变量组合与条件判断，来测试交互故事引擎的逻辑运算与功能稳定性 。
