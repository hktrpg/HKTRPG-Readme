# HKTRPG  官方使用教学

## **欢迎来到HKTRPG!**

HKTRPG是一个用于TRPG的网上工具，它允许你在不同平台上(如:网站，Discord，Line，Telegram，Plurk，Whatsapp)上进行掷骰。而且它不仅可以用于TRPG，还可以用于日常娱乐及群组管理。

### 使用理由

作为一个GM，由创建一个群组，保持良好的活跃气氛，教玩家开卡掷骰，再到跑团并向玩家展示各种信息，都是一项艰辛的工作。&#x20;

而使用HKTRPG，你可以在不同的平台中使用统一的指令，没有转换的烦恼并自带娱乐功能。

它的设计就是为了减少你的工作!&#x20;

### 5 分钟上手

1. **邀请** HKTRPG 到你的平台 → [邀请HKTRPG](kai-shi-shi-yong/yao-qing-hktrpg/README.md)
2. 输入 `1d100` 试试基本掷骰
3. 输入 `bothelp` 查看指令分类（或看 [指令速查表](kai-shi-shi-yong/zhi-ling-su-cha-biao.md)）
4. 建立 [角色卡](trpg-gong-neng/kai-shi-jin-hang-trpg/jiao-se-ka.md)：`.char add …` / `.char use 名字`
5. 需要隐藏结果时用 [暗骰](trpg-gong-neng/kai-shi-jin-hang-trpg/an-tou.md)：`dr` / `ddr` / `dddr`

{% hint style="info" %}
**小知识:** HKTRPG是原生的繁体中文程序，本来是为了跑CoC而开发，但现在独立支持十数种TRPG系统。觉得做得好，可以[Patreon支持](https://www.patreon.com/HKTRPG)。
{% endhint %}
