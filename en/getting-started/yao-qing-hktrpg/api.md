# API

Responses are JSON; requests use GET.\
Endpoint: [https://api.hktrpg.com/](https://api.hktrpg.com/)

Example: [https://api.hktrpg.com/?msg=1d100](https://api.hktrpg.com/?msg=1d100)

![Example in use](https://github.com/hktrpg/TG.line.Discord.Roll.Bot/blob/master/views/image/api.png?raw=true)

{% hint style="info" %}
Like the web version, this method cannot pass a user ID, so personal database features such as `.ra`, `.db`, and personal character sheets are not available.
{% endhint %}
