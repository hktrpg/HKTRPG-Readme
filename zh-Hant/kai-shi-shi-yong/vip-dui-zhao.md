# VIP 權益對照

查詢自己／群組等級：輸入 `.patreon level` 或 `.admin id`。

贊助與方案說明見 [開發支援](kai-fa-zhi-yuan.md)。

{% hint style="warning" %}
下表為文件整理用摘要（對應 Bot 內各模組的等級陣列）。伺服器可能調整上限；衝突時以 Bot 即時提示為準。
{% endhint %}

### 等級概念

* **VIP 0**：一般用戶
* **VIP 1+**：Patreon／Boost 等提升後的等級（個人或群組取較高者）
* 部分功能同時看「個人 VIP」與「群組 VIP」

### 常見功能上限（摘要）

| 功能 | VIP 0（約） | VIP 1～2（約） | VIP 3+（約） | 說明 |
| ---- | ----------- | -------------- | ------------ | ---- |
| 角色卡數量 | 4 | 20 | 30～99 | `.char` |
| 按鈕轉發 `.forward` | 4 | 20 | 30～99 | Discord |
| 自定義骰子 `.ra`（群組） | 30 | 200 | 500 | 群組關鍵字 |
| 自定義骰子（個人） | 2 | 200 | 500 | 個人關鍵字 |
| 資料庫 `.db` | 30 | 200 | 300 | |
| 儲存指令 `.cmd` | 30 | 200 | 300 | |
| 定時 `.at` | 5 | 25～50 | 200 | |
| 週期 `.cron` | 2 | 15～30 | 45～99 | |
| Discord 團錄匯出（週） | 1 | 20～40 | 40～99 | 另有冷卻 |
| StoryTeller 劇本數 | 3 | 10 | 100 | 見 StoryTeller 頁 |
| StoryTeller 同時遊戲 | 3 | 10 | 100 | |
| AI 對話／翻譯字數 | 較低 | 較高 | 更高／解鎖中級模型 | 見 AI 頁 |
